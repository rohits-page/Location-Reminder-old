Introduction

Next time you think "I should not forget to do [task] when I am at [location] between [date/time] & [date/time]", simply pull your phone out and set a location based reminder.
This APP will help you remind things based on location and time (optional). Be it reminding you to withdraw money out of your ATM next Sunday when you are close to your ATM or pickup the dry cleaning that you forgot the last few times you drove by. This APP will help will remind you when you are in close vicinity of your place of interest. Optionally you can configure the reminder to be active only within a specific time window.

APP can use both GPS and WiFi to assess current location. However it is recommended that GPS be turned ON for more accurate results.
You need to turn on either one of them for receiving notifications when you are in close vicinity of your place of interest.

 

How to set a reminder?

Select Location

There are two ways to select location

  • Enter Address
  • Select a point on Map

Enter Address

You may enter address and hit the Search icon on the screen. Upon successful search, you will see a marker on the screen. Tap on it.

Select Point on Map

You may tap on any point of interest on the displayed map area. A marker would appear on the screen. Tap on it.

 

Tip: You may turn on the My Location button if you would like the map to centered around your current location. Once turned ON, it may take a few seconds for the map to acquire your current location. It is advised that you turn OFF the MyLocation button once map is centered. It will help you conserve battery.

 

Enter Reminder Details

Once you tap on the Marker, you will be presented with a screen to enter details for the reminder.

Reminder details include:

  • Reminder Name: A name for this reminder. This is a mandatory field.
  • Description: You may use this field to enter What you want to be reminded of.
  • Address: The address will be pre filled. You will not be able to edit the address.
  • Radii: Sets a reminder for the selected address and the given radius. When the device detects that it has entered or exited the area surrounding the location, the reminder will be fired. If no value is entered it will set the default to 1Mile or 1KM, based on some of the default internal settings of your phone.
  • One Time Reminder: Checking this box will create a onetime reminder. If you would like this reminder to be triggered every time you are around the address, Un-check it.

    • Note: You can change the recurrence type after you have created the Reminder.

  • Timing Details: Checking this box will allow you to enter timing details. You can select the window within which the Reminder will be active.
    Leaving this box unchecked (or unchecking the box) will make the reminder time independent. It should be used for situations when you want to be reminded the next time you are the desired location.
    Checking the box will enable a few more controls. It allows you to select:
    • From Date/Time You can enter a Date/Time that marks the Start of reminder window. If selected Start Time is after the End Time, application will automatically set the End time to 24 hours from the selected start time.
    • End Date/Time You can enter a Date/Time that marks the End of reminder window.
    • All Day Event Checking this box automatically sets the time to mid night.

 

While saving a reminder, application checks for existing reminders with the same address. If it finds existing reminders for same address it will prompt with following 3 options.

  • Create New: Create the new reminder as specified.
  • Update Existing: All the existing Reminders for the selected Address will be presented. Select the ones you would like to modify.
  • Quit: Abort the reminder creation process.

Reminder Notification

Due to approximate nature of position estimation, a reminder will be created when you approximately pass the area of interest.

Based on the default notification settings on your device, notification may include sound, vibration, flashing LED or a combination.

You will see a notification with the Title Location Based Reminder in the Notification tray of your device.

It will also contain the name of the Reminder.

In case there are multiple Reminders, set for the area you are in, you may see multiple notifications in the tray.

Hence it is important to give meaningful names to the Reminders. It will help you select the notification of interest.

Upon selecting the notification of interest Reminder details will be provided.

If you wish to Edit/Delete the Reminder; you can do so from here.

Managing Reminder

This section refers to Viewing, Editing and Deleting existing Reminders.

1.     Navigate to main screen with the Map View.

2.     Click the Menu button of your device.

3.     Select View.

If no Reminders exist, an appropriate message will be displayed.

Else, a list of existing Reminders will be presented.

You may scroll through the list to select the desired Reminder.

4.     Long Press the Reminder you wish to view/edit/delete.

5.     The selected Reminder will be presented in View mode and options to Edit or Delete will be provided. Click on appropriate Button. You may press OK if you wish to exit the Reminder details screen.

 

From the list view screen, Press Back key on your device to return to Map View.

Purchasing APP

The Free/trial version of the APP has the following restrictions:
  • Only 20 Reminders.
  • Only 1 Time Alerts.
The trial version will not allow Recurring Alerts. You may purchase the Key to unlock the APP and explore the full potential.
Click here to purchase the key (Location Reminder Key Plugin)
Once you download the 'Location Reminder Key Plugin', it will appear as 'Key Plugin(Loc Reminder)' in your APP tray.
Steps to Install the Key
  • Close the Location Reminder APP
  • Launch the 'Key Plugin (Loc Reminder)' APP from APP Tray. (You will see no change on the screen.)
  • Launch the Location Reminder APP

Reinstall APP

In an event that you have to reinstall the Application(s), you will need to download both the APP's. It is recommended that you download the 'Location Reminder APP' first and then the 'Reminder Location Key Plugin APP'.
Restart your device after you download the APP(s) prior to launching them.
If you try to reinstall the trial version without purchasing the Key, your reminders may not work as expected.

Clean

There is a Menu option 'Clean' that will Delete all the user created Reminders.
Only the Full version of the APP will delete any reminders with name 'DUMMY_LOCATION_REMINDER'. These reminders will be created by the APP under special circumstances.
